# Auto Sort Gmail Attachments to Google Drive using Make

Automatically save Gmail attachments to organized Google Drive folders using Make, eliminating manual downloads and file organization.

---

# 📖 Overview

This automation monitors incoming Gmail emails, detects attachments, and automatically uploads them to the appropriate Google Drive folders.

By automating file organization, businesses can reduce manual work, keep documents organized, and improve team productivity.

---

# 💼 Business Problem

Downloading email attachments manually and organizing them into folders is repetitive, time-consuming, and prone to human error.

As the number of emails increases, managing files becomes inefficient and difficult to maintain.

---

# 💡 Solution

Built a Make automation that:

- Monitors incoming Gmail emails
- Detects email attachments
- Identifies the appropriate destination folder
- Uploads attachments to Google Drive automatically
- Organizes files without manual intervention

The workflow ensures every attachment is stored in the correct location immediately after receiving the email.

---

# 🔄 Workflow

```text
New Gmail Email
        │
        ▼
Detect Attachment
        │
        ▼
Make
        │
        ▼
Find Destination Folder
        │
        ▼
Upload File to Google Drive
```
